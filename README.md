## 🚀 Installation & Usage

### Requirements
- [NVM (Node Version Manager)](https://github.com/nvm-sh/nvm) installed
- Visual Studio Code 1.97.0 or higher

### Installation
1. Search "NVM Node Switch" in VS Code extensions marketplace
